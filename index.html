About Me

I am a dedicated Web Developer and Java Full Stack Developer with hands-on experience in designing and developing user-friendly web applications. On the frontend, I have created interactive projects like a Basic Calculator, an Age Calculator, and an Image Slider using HTML, CSS, and JavaScript, which showcase my understanding of dynamic and responsive UI design. On the backend, I’m currently working on two major ongoing projects. The first is CardaConnect, an online cardamom trading platform that facilitates seamless buying and selling between vendors and customers, with features like stock uploads, order management, and transport tracking. The second is a Student Placement Appointment Management System, which allows students to easily schedule campus placement interviews with company-specific preferences. These projects reflect my growing expertise in full-stack development, database integration, and building real-world solutions with a focus on usability and performance.
